# Slim calibration data

This directory carries a **slim copy** of the per-sample review data
shipped with the Cerebras challenge under `Evals/`. The full data is
~920 MB (LFS-backed); the slim form below is **~200 KB** and is what
the pruners actually need.

## What was kept vs. dropped

| Field | Kept? | Why |
|---|---|---|
| `index` | ✅ | Joins predictions ↔ reviews ↔ upstream `live_code_bench` / `aa_lcr` samples. |
| `model` | ✅ | Three reference models per benchmark — required for difficulty + discrimination calculations. |
| `score.value.pass` / `score.value.acc` | ✅ | The single number the pruner consumes per (sample, model). |
| `metadata.{difficulty,platform,contest_date,tags,question_id}` (LCB) | ✅ | Used for diversity stratification + temporal-leakage filtering. |
| `metadata.{num_documents,context_tokens,category}` (AA-LCR) | ✅ | Used for context-length stratification. |
| `prediction_len` (AA-LCR) | ✅ | Cheap proxy for response-confidence under judge noise. |
| Full `model_output.choices[].message.content` | ❌ | Full model completions — megabytes per sample, never read by a pruner. |
| Full `extracted_prediction` | ❌ | Same reason. |
| Reasoning traces | ❌ | Same reason. |

The full review files are available in the challenge repo under
`Evals/Part 1/` and `Evals/MMMU/`. Use those if you want to inspect raw
completions; use this slim copy for pruning.
